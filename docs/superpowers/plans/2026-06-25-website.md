# Proctor website + discovery — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans (inline) to implement task-by-task. This is a frontend/content build, so "verify" = `npm run build` succeeds + visual/Lighthouse check (not unit tests). Use the **frontend-design** skill when building the landing (Task 4). Steps use `- [ ]`.

**Goal:** A fast, SEO-strong static site (landing + Starlight docs + one cornerstone post) for Proctor at `proctorbench.dev`, deployed free, plus the discovery assets.

**Architecture:** Astro + Starlight in `site/` (in the Proctor repo, `website` branch). Custom Astro landing at `/`, Starlight docs at `/docs/*` sourced from the existing repo markdown, an Astro content-collection blog at `/blog`. Static output to `site/dist`, hosted on Cloudflare Pages; canonical domain `proctorbench.dev` (CNAME attached after registration; ship on `*.pages.dev` first).

**Tech Stack:** Astro 5, @astrojs/starlight, @astrojs/sitemap, Node 22 / npm 10.

### Task 2: Site config — domain, SEO defaults, sitemap, theme tokens

**Files:** Modify `site/astro.config.mjs`; create `site/src/styles/custom.css`.

- [ ] **Step 1: Configure `astro.config.mjs`**

Set `site: 'https://proctorbench.dev'`, add the sitemap integration, configure Starlight (title "Proctor", social link to `https://github.com/dylanp12/proctor`, the sidebar from Task 3, the custom CSS, and `head` entries for default OG/Twitter meta). Concretely:

```js
import { defineConfig } from 'astro/config';
import starlight from '@astrojs/starlight';
import sitemap from '@astrojs/sitemap';

export default defineConfig({
  site: 'https://proctorbench.dev',
  integrations: [
    starlight({
      title: 'Proctor',
      description: 'Turn AI coding-agent benchmark runs into signed, independently verifiable integrity bundles.',
      social: { github: 'https://github.com/dylanp12/proctor' },
      customCss: ['./src/styles/custom.css'],
      sidebar: [
        { label: 'Overview', slug: 'docs/overview' },
        { label: 'Quickstart', slug: 'docs/quickstart' },
        { label: 'How it works', slug: 'docs/how-it-works' },
        { label: 'Bundle spec', slug: 'docs/bundle-spec' },
        { label: 'Honest scope', slug: 'docs/honest-scope' },
        { label: 'FAQ', slug: 'docs/faq' },
        { label: 'Roadmap', slug: 'docs/roadmap' },
        { label: 'Example bundle', slug: 'docs/example-bundle' },
      ],
    }),
    sitemap(),
  ],
});
```

- [ ] **Step 2: Theme tokens** — in `site/src/styles/custom.css`, set Starlight accent + a dark, technical, monospace-accented palette (one restrained accent; the `compromised` verdict as the motif color). Keep minimal.
